# meshmapper-android
MeshMapper for Android is a portal to meshmapper.net, providing access to the wardriving dashboard and mesh maps, including the ability to pin a city.

## Install

To install, download the APK from Releases, or clone the project and build from source (super easy to do in Android Studio).
